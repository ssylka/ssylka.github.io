📦 Inventory Management

Веб-приложение для ведения инвентаризаций любого типа: оргтехника, книги, документы. Пользователи создают собственные шаблоны инвентаризаций с гибкой структурой и управляют элементами внутри них.

  • Конструктор пользовательских ID: drag & drop из блоков (текст, случайные числа, GUID, дата, последовательность) с live-превью и контролем уникальности на уровне БД
  • Настраиваемые поля 5 типов с управлением видимостью в таблице
  • Разграничение доступа: публичные и приватные инвентаризации, страница администратора
  • OAuth-вход через Google и GitHub, полнотекстовый поиск на каждой странице
  • Обсуждения в реальном времени, лайки, автосохранение с оптимистичными блокировками
  • Статистика по элементам, локализация, светлая/тёмная тема

⚡ Power Automate Автоматизация

Сервис поддержки: с любой страницы можно создать тикет, указав тему и приоритет (High / Average / Low). Приложение формирует JSON — автор, инвентаризация, ссылка на исходную страницу, приоритет — и загружает его в облачное хранилище. Облачный flow в Power Automate срабатывает на новый файл, читает JSON, рассылает оформленные письма администраторам и отправляет push-уведомление в мобильное приложение Power Automate.

🟣 Odoo ERP

Внешний API с доступом к агрегированным результатам инвентаризаций по api-токену (токен генерируется отдельно для каждой инвентаризации). Собственное приложение Odoo импортирует по токену название, набор полей и агрегаты — среднее/мин/макс для чисел, самые популярные значения для текста — и работает как read-only viewer со списком импортированных инвентаризаций и детальным просмотром каждой.
